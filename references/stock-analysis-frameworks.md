## 价值投资分析框架

### 格雷厄姆安全边际估值

#### Graham Number（格雷厄姆数）

```
Graham Number = √(22.5 × EPS × BVPS)

参数说明：
- EPS：稀释每股收益（通常使用TTM或最近年度）
- BVPS：每股净资产
- 22.5：格雷厄姆的经验系数（15×PE × 1.5×PB = 22.5）
```

**判断标准**：
- 当前股价 < Graham Number → 有安全边际，可考虑买入
- 当前股价 > Graham Number × 1.2 → 估值偏高
- 当前股价 > Graham Number × 1.5 → 估值泡沫

#### DCF估值（现金流折现）

```
内在价值 = Σ[FCF_t / (1+WACC)^t] + TV / (1+WACC)^n

简化公式（永续增长模型）：
DCF = FCF × (1+g) / (WACC - g)

参数说明：
- FCF：最近年度自由现金流
- g：永续增长率（通常 2-4%）
- WACC：加权平均资本成本
  WACC = Ke × E/(D+E) + Kd × (1-T) × D/(D+E)
  其中：Ke = Rf + β × ERP

三场景：
- 保守：g=2%, WACC=10%
- 中性：g=3%, WACC=9%
- 乐观：g=4%, WACC=8%
```

#### 安全边际计算

```
安全边际 = (DCF中性估值 - 当前股价) / DCF中性估值 × 100%

安全边际 > 40% → 深度价值
安全边际 20-40% → 合理价值
安全边际 < 20% → 估值偏高
安全边际 < 0 → 估值泡沫
```

### 巴菲特护城河分析

#### 五维度评估

| 维度 | 评估问题 | 评级标准 |
|------|---------|---------|
| **品牌护城河** | 品牌能否带来溢价？客户是否主动选择？ | 宽：溢价>10%；窄：溢价5-10%；无：无溢价 |
| **转换成本** | 客户迁移到竞品的实际成本有多高？ | 宽：高迁移成本；窄：中等；无：低迁移成本 |
| **网络效应** | 用户增加是否带来产品/服务价值指数增长？ | 宽：指数增长；窄：线性增长；无：无网络效应 |
| **成本优势** | 能否以低于竞品的方式提供服务？ | 宽：持续低成本；窄：阶段性低成本；无：无成本优势 |
| **监管护城河** | 是否具有政策保护或特许经营权？ | 宽：强监管保护；窄：一般监管；无：无监管壁垒 |

#### 护城河评级

- **宽护城河**：5个维度中有3个以上达到"宽"
- **窄护城河**：2-3个维度达到"宽"，其余不是"无"
- **无护城河**：有2个以上维度为"无"

### 霍华德·马克斯周期思维

#### 周期阶段识别

| 阶段 | 特征 | 估值信号 | 情绪信号 |
|------|------|---------|---------|
| **早期** | 估值低、业绩差、资本开支谨慎 | PE历史低位、分位数<20% | 悲观、负面新闻多 |
| **中期** | 估值修复、业绩增长、机构增配 | PE历史中位、分位数20-60% | 乐观、分析师上调评级 |
| **过热** | 估值泡沫、业绩顶峰、资本开支过度 | PE历史高位、分位数>80% | 狂热、IPO疯狂 |
| **衰退** | 估值压缩、业绩下滑、去杠杆 | PE下降通道 | 恐慌、并购减少 |

#### 市场情绪指标

| 指标 | 低位信号 | 高位信号 |
|------|---------|---------|
| PE分位数 | <20% | >80% |
| 股息率 vs 债券收益率 | 股息率较高 | 债券收益率较高 |
| 基金发行规模 | 发行困难 | 爆款频发 |
| 融资余额 | 持续下降 | 持续攀升 |
| VIX指数 | >30 | <15 |

## 技术分析框架

### 道氏理论趋势分析

#### 趋势定义

| 趋势类型 | 时间框架 | 特征 |
|---------|---------|------|
| 主要趋势 | 6个月-数年 | 牛市（高点低点依次抬高）或熊市（高点低点依次降低） |
| 次要趋势 | 3周-3个月 | 主要趋势的回调（熊市反弹或牛市回调），通常回撤1/3-2/3 |
| 短暂趋势 | <3周 | 日内或短期波动，通常是噪音 |

#### 趋势确认信号

**上涨趋势确认**：
1. 价格突破前一个高点
2. 回调低点高于前一个低点
3. 成交量在上涨时放大，下跌时缩量

**下跌趋势确认**：
1. 价格跌破前一个低点
2. 反弹高点低于前一个高点
3. 成交量在下跌时放大，反弹时缩量

#### 支撑位与阻力位

```
支撑位：过去价格多次反弹的低点
阻力位：过去价格多次遇阻的高点

计算方法：
- 近期低点/高点
- 均线位置（50日、200日）
- 整数关口
- 历史成交密集区
```

### 艾略特波浪分析

#### 波浪基本规则

1. 推动浪(1,3,5)与主趋势方向一致
2. 浪2不破浪1起点
3. 浪4不破浪1终点
4. 第3浪通常是最长的浪
5. 第4浪不会进入第1浪的价格区间

#### 波浪目标位计算

```
推动浪目标：
- 2浪 = 1浪 × 0.382 / 0.500 / 0.618
- 3浪 = 1浪 × 1.618 / 2.618
- 4浪 = 3浪 × 0.382 / 0.500 / 0.618
- 5浪 = 3浪 × 0.618 / 1.618

如果5浪延伸：
- 5浪 = 1浪 × (1-3浪比例)

调整浪目标（锯齿形）：
- C浪 = A浪 × 0.618 / 1.000 / 1.618
```

#### 波浪置信度

| 置信度 | 条件 |
|-------|------|
| 高 | 满足所有基本规则、成交量确认、与基本面匹配 |
| 中 | 满足基本规则、但某些条件不明确 |
| 低 | 存在违反规则的情况、波浪计数不清晰 |

### 欧奈尔CAN SLIM分析

#### 七维度评分

| 字母 | 指标 | 优秀标准 | 1-10评分标准 |
|------|------|---------|-------------|
| **C** | 季度EPS增长 | >20%同比增长 | 1:<0%, 5:20%, 10:>50% |
| **A** | 年度EPS增长 | >25%（3-5年） | 1:<0%, 5:25%, 10:>50% |
| **N** | 新产品/管理层 | 有实质性创新 | 1:无，5:一般，10:重大创新 |
| **S** | 供需 | 流通股减少或机构增配 | 1:流出，5:持平，10:大幅增配 |
| **L** | 相对强度 | RS排名行业前20% | 1:行业末位，5:行业中游，10:行业第一 |
| **I** | 机构持仓 | 顶级基金新进或增配 | 1:无，5:一般，10:顶级基金大幅增配 |
| **M** | 大盘方向 | 与大盘同步或跑赢 | 1:跑输，5:同步，10:明显跑赢 |

#### 综合评级

- **90-100分**：超级领导者，买入
- **70-89分**：优秀，可以买入
- **50-69分**：中等，关注
- **30-49分**：较弱，谨慎
- **<30分**：差，避免

### 技术指标参考

#### RSI（相对强弱指数）

```
RSI = 100 - (100 / (1 + RS))
RS = 平均涨幅 / 平均跌幅

参数：14日（常用）

判断标准：
- RSI > 70：超买，可能调整
- RSI < 30：超卖，可能反弹
- RSI 50：多空平衡
```

#### MACD

```
MACD线 = 12日EMA - 26日EMA
信号线 = MACD的9日EMA
柱状图 = MACD线 - 信号线

参数：12, 26, 9

判断标准：
- MACD > 0：多头市场
- MACD < 0：空头市场
- 金叉（MACD上穿信号线）：买入信号
- 死叉（MACD下穿信号线）：卖出信号
```

#### 布林带

```
中轨 = 20日均线
上轨 = 中轨 + 2×标准差
下轨 = 中轨 - 2×标准差

参数：20日，2倍标准差

判断标准：
- 价格触及上轨：可能超买
- 价格触及下轨：可能超卖
- 布林带收窄：波动率降低，可能突破
- 布林带开口：波动率增加
```

#### 均线系统

| 组合 | 信号 |
|------|------|
| 50日均线 > 200日均线 | 黄金交叉（买入信号） |
| 50日均线 < 200日均线 | 死亡交叉（卖出信号） |
| 价格 > 200日均线 | 长期趋势向上 |
| 价格 < 200日均线 | 长期趋势向下 |

## 分析报告语言规范

### 必须使用中文
- 报告标题、章节标题
- 核心摘要、风险提示
- 页脚、日期标注
- 评价性用语

### 可使用英文
- 专业术语：PE、PB、EPS、DCF、RSI、MACD、ATR、FCF
- 公司/产品名：Apple、Microsoft、NVIDIA
- 公式和计算过程

### 数据标注
- 所有数据必须标注来源
- 实时数据注明采集时间
- 估算数据注明假设条件
